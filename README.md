# Aura Skin - Tienda online de skincare

## Descripción

Tienda online para marca de productos naturales de skincare. Catálogo con filtros por categorías (limpiadores, sérums, hidratantes, protectores solares, mascarillas, accesorios), carrito de compras interactivo y envío de pedidos formateado por WhatsApp.

## El problema que resuelve

La marca vendía únicamente por redes sociales, lo que dificultaba el seguimiento de pedidos y la organización. Esta tienda online profesionaliza las ventas y automatiza el proceso de pedidos.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
